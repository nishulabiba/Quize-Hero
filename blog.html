filter() and map() return a new collection, find() returns a single element, and foreach() returns nothing but allows an operation to be performed on each element in the collection. But I think the map() and filter() has advantages over for() and foreach(). Advantages: No need to write too much of code and very clear understanding and better performance compare to iteration.
